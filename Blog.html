What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM provides a structured representation of the document as a tree of objects, and it defines methods to access these objects.

How do you select an element from the DOM?

I can select an element from the DOM using methods like

document.getElementById()

,

document.getElementsByClassName()

,

document.querySelector()

, or

document.querySelectorAll()

. These methods help me find elements by ID, class name, tag name, or a CSS selector.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element to manage events triggered by its child elements. This approach is useful because it reduces the number of event listeners needed, improves performance, and allows dynamic elements added later to inherit event handling automatically.

How do you manipulate an element's attributes and styles using the DOM?

I can manipulate an element's attributes using methods like

setAttribute()

and

getAttribute()

. To change styles, you can modify the element's

style

property directly, or add/remove classes using

classList.add()

,

classList.remove()

, or

classList.toggle()

.
